Blocky Bounce

Blocky Bounce is my first ever iOS game. It was done to learn more about iOS development, as well as game development. Block Bounce is a simple acrade style game where you can compete with your friends to get the highest score using the in game leader-board. The small scale of the game allowed me to finish the game by myself, but was complex enough to teach me a lot. Blocky Bounce was made using Unity 3D, and was scripted in C#.

You can download the game for free on the App Store here:

https://itunes.apple.com/nz/app/blocky-bounce/id1346412974?mt=8
